# Spring Boot Frontend Starter

![CodeQL](https://github.com/wlindner/spring-boot-frontend-starter/workflows/CodeQL/badge.svg)

## Install Dependencies

```
./gradlew npm_install
```

## Run App

Start the frontend

```
./gradlew npm_run_start
```

Start the backend

```
./gradlew bootRun
```

## Build App

Build the frontend

```
./gradlew npm_run_build
```

Build the backend

```
./gradlew build
```
